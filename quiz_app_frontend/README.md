# Quiz App Frontend (LightningJS / Blits)

Ocean Professional themed simple quiz app with a safe Intro → Quiz → Results flow.

## Run locally
1. npm install
2. npm run dev
3. Open http://localhost:3000

The app uses static quiz data and does not require any backend.

## Environment variables
Optional Vite variables (set in .env if needed):
- VITE_API_BASE
- VITE_BACKEND_URL
- VITE_FRONTEND_URL
- VITE_WS_URL
- VITE_NODE_ENV
- VITE_NEXT_TELEMETRY_DISABLED
- VITE_ENABLE_SOURCE_MAPS
- VITE_PORT
- VITE_TRUST_PROXY
- VITE_LOG_LEVEL
- VITE_HEALTHCHECK_PATH
- VITE_FEATURE_FLAGS
- VITE_EXPERIMENTS_ENABLED

By default, the dev server runs on port 3000.
